The Problem

Farmers can't access science-based knowledge because it's not in their language.

  • WUR holds world-class agricultural research ‚Äî locked in English/Dutch
  • 80% of smallholder farmers speak Swahili, French, Arabic
  • Current cloud-based tools create data security risks
  • Critical gaps: climate-resilient seeds, pest control, market info
Visual: Map of Sub-Saharan Africa with red dots where knowledge doesn't reach

Why On-Premise?

Data isn't just private ‚Äî it's sacred.

  • Sensitive farmer records: land use, income, yields
  • GDPR-like regulations emerging across Africa
  • No external cloud dependencies = full control
  • Trust + compliance + sustainability are non-negotiable
Visual: Shield icon protecting a phone with local-language UI

Our Solution

An AI advisor that works without internet.

  • Runs on a single laptop ‚Äî no cloud, no login
  • Answers questions in Swahili, French, Arabic
  • Powers local advisory through Mistral 7B + FAISS + Ollama

Multilingual RAG Strategy

Ask in Swahili. Get answer from English documents.

Current Capability: English, Spanish, French queries
Approach: Multilingual embeddings handle cross-language retrieval
Quality: English documents answer non-English questions accurately

RAG + ML Integration Patterns

  • Sequential: ML identifies risk ‚Üí RAG explains how to respond
  • Parallel: Both process query ‚Üí results merged
  • Conditional: Query complexity determines routing
  • Feedback Loop: RAG findings improve ML feature selection
